# UIDE
# Introducción

Este proyecto consiste en el desarrollo de un **generador de contraseñas seguras** implementado en Python. La aplicación permite crear contraseñas aleatorias con tres niveles de seguridad (8, 12 y 16 caracteres), asegurando la inclusión de letras mayúsculas, letras minúsculas, números y caracteres especiales para mejorar la protección de las credenciales de los usuarios.

## Descripción del problema

Uno de los problemas más comunes en la seguridad informática es el uso de contraseñas débiles o repetidas, lo que incrementa el riesgo de accesos no autorizados y compromete la información personal. Este proyecto busca ofrecer una solución sencilla que permita generar contraseñas robustas de manera automática, promoviendo buenas prácticas de seguridad.

## Relación con los contenidos de la asignatura

El desarrollo del proyecto integra los conocimientos adquiridos durante las cuatro unidades de la asignatura, aplicando análisis del problema, diseño básico de software, funciones, estructuras condicionales (`if`, `elif`, `else`), ciclos (`while` y `for`), listas, además del uso de las bibliotecas estándar `random` y `string`. También se utilizó GitHub para el control de versiones del código.

## Explicación del sistema desarrollado

El programa funciona mediante un menú interactivo en consola donde el usuario selecciona el nivel de seguridad deseado. A partir de esa selección, el sistema genera una contraseña aleatoria garantizando que contenga al menos una letra mayúscula, una letra minúscula, un número y un carácter especial. Posteriormente, los caracteres se mezclan aleatoriamente antes de mostrar el resultado, permitiendo al usuario generar nuevas contraseñas hasta decidir finalizar el programa.
